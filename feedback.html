Signaler une erreur de traduction

Il est possible que tu aies rencontré une erreur dans la traduction d'un mot.
Pour que je puisse corriger l'erreur et offrir une meilleure expérience de révision à tout le monde, suis les étapes ci-dessous.

Pendant un entraînement

Si tu rencontres une erreur de traduction pendant un entraînement, clique sur le bouton intitulé « Signaler une erreur de traduction ».

Celui-ci ouvrira une boîte de dialogue contenant deux champs de texte.
Le premier contient le mot présent dans la question actuelle ainsi que ses alternatives, si il en dispose.
Le second contient la réponse à la question actuelle ainsi que ses alternatives, si elle en dispose.

Tu peux à présent corriger l'erreur que tu as repérée. Une fois fait, clique sur le bouton « Envoyer ».
Et voilà, le tour est joué !


Hors d'un entraînement

Il n'existe pas encore de manière simple et intuitive d'envoyer un signalement en dehors de la page d'entraînement. J'y travaille, donc pour le moment il faut se contenter de rapporter l'erreur via la page GitHub du site web. N'aies pas peur, je vais te guider pas à pas !


 1. Crée un compte GitHub

GitHub est une plateforme que les développeurs utilisent pour partager leur code. Ne t'inquiètes pas, c'est sans danger !
Crée un compte en te rendant sur la page d'accueil de GitHub.
Renseigne un nom d'utilisateur, une adresse mail et un mot de passe.


 2. Accède à la page GitHub du projet

Rends-toi sur la page GitHub du projet. Tu peux alors voir le code qui fait fonctionner le site sur lequel tu te trouves actuellement. (C'est trop cool n'est-ce pas ?)


 3. Clique sur « Issues » puis « New issue »

Une issue c'est un rapport d'erreur ou une suggestion. C'est grâce à ces rapports que les développeurs sont avertis de la présence d'un problème dans leur code.


 4. Décris le problème

Dans la description, précise de quel mot il s'agit avec son orthographe actuel et l'orthographe corrigé.


 5. Clique sur « Submit new issue »

Et voilà, le tour est joué ! Je pourrais ainsi investiguer le problème et le corriger au plus vite.
