À propos de Vokabeln!

Un projet collaboratif

Vokabeln! est un projet collaboratif. C'est-à-dire que le site évolue grâce aux retours de ses utilisateurs.
En signalant une erreur de traduction ou en améliorant le code du site internet, tu aides à améliorer les révisions d'autres élèves.

Si tu souhaites contribuer au projet, rends-toi sur la page GitHub du projet.

Récolte de données analytiques anonymes

Vokabeln! récolte des données analytiques anonymes par l'intermédiaire de Google Analytics. Ces données comprennent notamment le nombre de visites par page du site. Celles-ci sont analysées dans l'unique intérêt de pouvoir offrir une meilleure expérience utilisateur.
